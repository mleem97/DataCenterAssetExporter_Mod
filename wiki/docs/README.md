# Where the wiki Markdown lives

The Docusaurus app in this folder uses **`path: ../docs`** (see `wiki/docusaurus.config.js`). All browsable Markdown/MDX sources are under the **repository root** [`docs/`](../docs/), not under `wiki/docs/`.

| Content | Path |
|--------|------|
| Full **`docs/`** map (curated vs import vs stubs) | [`docs/README.md`](../docs/README.md) |
| Legacy GitHub Wiki import | [`docs/wiki-import/`](../docs/wiki-import/) |
| Curated monorepo stubs (not the GitHub Wiki) | [`docs/wiki/README.md`](../docs/wiki/README.md) |
| Sync `.wiki` → `docs/wiki-import/` | `npm run wiki:sync` from **`wiki/`** (requires `.wiki/` at repo root) |
| Split DE/EN pairs into default locale + `wiki/i18n/de/...` | `npm run wiki:normalize-i18n` (optional `--dry-run`) |
| **Both** (after updating `.wiki`) | `npm run wiki:refresh` |

The **Legacy wiki import** section in the sidebar lists the full `docs/wiki-import/` tree (autogenerated), with audience folders ordered via `_category_.json` positions.
